# مسئله رمزگشایی پیام

## عنوان: رمزگشایی پیام

یک پیام رمزگذاری‌شده به صورت رشته‌ای از کلمات که با فاصله از هم جدا شده‌اند، به شما داده می‌شود. قوانین رمزگذاری به این صورت است:

- هر کلمه **عددی** (مانند `Zero`، `One`، ...، `Nine`) با رقم متناظر خود (`0` تا `9`) جایگزین می‌شود.
- هر کلمه **غیرعددی** با **حرف اول** آن کلمه جایگزین می‌شود.

برنامه‌ای بنویسید که چنین رشته‌ای را گرفته و خروجی نهایی (رشته رمزگشایی‌شده) را چاپ کند.

### ورودی نمونه

```
Whiskey Hotel Four Tango Dash Alpha Romeo Three Dash Yankee Oscar Uniform Dash Sierra One November Kilo India November Golf Dash Four Bravo Zero Uniform Seven
```

### خروجی مورد انتظار

```
WH4T D AR3 D YOU D S1NK ING D 4B0U7
```

> **نکته:** خروجی بالا بر اساس الگوریتم توضیح‌داده‌شده به دست آمده است.

# Message Decoding Problem

## Title: Message Decoding

You are given an encoded message as a string of space-separated words. The encoding rules are as follows:

- Each **number word** (e.g., `Zero`, `One`, ..., `Nine`) is replaced with its corresponding digit (`0` to `9`).
- Each **non‑number word** is replaced with its **first letter**.

Write a program that takes such a string and prints the final decoded string.

### Sample Input

```
Whiskey Hotel Four Tango Dash Alpha Romeo Three Dash Yankee Oscar Uniform Dash Sierra One November Kilo India November Golf Dash Four Bravo Zero Uniform Seven
```

### Expected Output

```
WH4T D AR3 D YOU D S1NK ING D 4B0U7
```

> **Note:** The output above is obtained by applying the described algorithm.
