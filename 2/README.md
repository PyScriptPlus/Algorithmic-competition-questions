# مسئله چاپ تکراری Linux

## عنوان: چاپ پشت سر هم

تیم لینوکس از شما می‌خواهد برنامه‌ای بنویسید که یک عدد از ورودی بگیرد و به تعداد آن، کلمه `Linux` را پشت سر هم (بدون فاصله) چاپ کند.

### ورودی

در خط اول ورودی عدد `n` می‌آید.

محدودیت:  
1 <= n <= 100

### خروجی

خروجی شامل یک رشته است که در آن `n` بار کلمه `Linux` پشت سر هم قرار گرفته است.

### مثال

**ورودی نمونه ۱:**
```
3
```
**خروجی نمونه ۱:**
```
LinuxLinuxLinux
```

**ورودی نمونه ۲:**
```
1
```
**خروجی نمونه ۲:**
```
Linux
```

# Repeated Linux Printing Problem

## Title: Concatenated Repetition

The Linux team asks you to write a program that reads an integer and prints the word `Linux` repeated that many times, with no spaces in between.

### Input

The first line of input contains an integer `n`.

Constraints:  
1 <= n <= 100

### Output

Output a single string consisting of the word `Linux` repeated `n` times consecutively.

### Examples

**Sample Input 1:**
```
3
```
**Sample Output 1:**
```
LinuxLinuxLinux
```

**Sample Input 2:**
```
1
```
**Sample Output 2:**
```
Linux
```
